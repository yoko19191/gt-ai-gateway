# v1.0/v1.1 前端产品文档

## 项目概述

完成 AI 网关的基础管理功能，包括登录认证、用户管理、供应商管理和模型管理的前端界面。

---

## 功能模块

### 登录认证（v1.0）

**界面元素**
- 登录表单
  - Token 输入框（密码类型）
  - 登录按钮

**交互流程**
1. 用户输入 Token
2. 点击登录按钮
3. 验证成功后进入管理后台，失败显示错误提示

---

### 用户管理（v1.0）

**界面元素**
- 用户列表页面
  - 表格展示：ID、用户名、Token（部分隐藏）、类型、创建时间
  - 搜索框（按用户名搜索）
  - 筛选器（按用户类型筛选）
  - "新建用户"按钮
  - 分页器

- 新建用户对话框
  - 用户名输入框
  - Token 输入框（密码类型，可选）
  - 类型选择器（normal/admin）
  - 保存/取消按钮

- 用户详情页面
  - 用户名
  - Token（可复制完整 Token）
  - 类型
  - 创建时间、更新时间

**交互流程**
1. 用户列表页展示所有用户
2. 点击"新建用户"打开创建对话框
3. 输入用户信息并保存
4. 点击用户名查看详情

---

### 供应商管理（v1.1）

**界面元素**
- 供应商列表页面
  - 表格展示：ID、类型、名称、API 格式、URL、状态、创建时间
  - 搜索框（按名称搜索）
  - 筛选器（按类型、API 格式筛选）
  - "新建供应商"按钮
  - 分页器

- 新建供应商对话框
  - 类型选择器（openai/anthropic/aliyun/deepseek/google/other）
  - 名称输入框
  - Token 输入框（密码类型）
  - URL 输入框
  - API 格式选择器（openai/anthropic/google）
  - 保存/取消按钮

- 供应商详情页面
  - 完整信息展示
  - 关联模型列表

**交互流程**
1. 供应商列表页展示所有供应商
2. 点击"新建供应商"打开创建对话框
3. 输入供应商信息并保存
4. 点击供应商名称查看详情

---

### 模型管理（v1.1）

**界面元素**
- 模型列表页面
  - 表格展示：ID、模型名称、供应商名称、状态、创建时间
  - 搜索框（按模型名搜索）
  - 筛选器（按供应商筛选）
  - "新建模型"按钮
  - 分页器

- 新建模型对话框
  - 供应商选择器（下拉选择已存在的供应商）
  - 模型名称输入框
  - 启用状态开关
  - 保存/取消按钮

**交互流程**
1. 模型列表页展示所有模型
2. 点击"新建模型"打开创建对话框
3. 选择供应商并输入模型名称
4. 保存后返回列表

---

### 仪表盘（v1.0）

**界面元素**
- 系统状态卡片
  - 运行状态（正常/异常）
  - 系统版本
  - 数据库连接状态
