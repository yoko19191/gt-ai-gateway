# Serverless AI Gateway - 前端开发路线图

## 项目概述

构建 Serverless AI Gateway 的前端管理界面，提供用户管理、供应商管理、模型管理、请求监控和 API 测试等功能。

---

## 版本概览

### v1.0: 基础框架 + 用户管理

完成基础框架搭建和用户管理功能，建立项目骨架和认证体系：

- Vue 3 + TypeScript + Vite 项目搭建
- Ant Design Vue 集成
- Vue Router 路由配置 + Pinia 状态管理
- Axios 封装（请求/响应拦截器）+ 全局错误处理
- 侧边栏 + 主内容区域布局 + 顶部导航栏
- Token 登录认证系统
- 用户列表展示、新建用户、用户详情查看
- Token 显示/隐藏切换、用户类型筛选、按用户名搜索、分页功能
- 系统运行状态展示

### v1.1: 供应商 + 模型管理

完成供应商和模型管理功能，补齐核心管理能力：

- 供应商列表、新建、详情查看、编辑功能
- Token 显示/隐藏切换、按类型/格式筛选、按名称搜索、分页
- 模型列表、新建、详情查看、编辑功能
- 供应商关联选择、按供应商筛选、按模型名搜索、分页
- 加载状态优化（骨架屏）、操作提示优化、表格空状态优化

### v2.0: 请求记录

实现请求记录查看和详情分析功能，方便追踪系统使用情况：

- 请求记录列表展示、请求详情查看
- 请求数据/响应数据 JSON 展示（语法高亮）
- 复制 JSON 数据功能
- 按状态筛选（success/failed）、按时间范围筛选
- 按用户名/模型名搜索、分页功能、自动刷新功能

### v2.1: 仪表盘 + API 测试

增加仪表盘统计和 API 测试工具，提升数据可视化和开发调试体验：

- 今日请求总数、成功率、活跃用户数、活跃模型数统计
- 最近请求快捷入口、数据自动刷新（30秒）、刷新控制按钮
- API 格式选择器（OpenAI/Anthropic）、模型选择器
- 消息输入区域（支持多轮对话）、参数配置面板
- 发送按钮、响应展示区域、流式响应支持（SSE）
- 清除/重新发送功能、测试历史记录

### v3.0: 删除 + 导出

增加删除功能和数据导出功能，提升系统完整性和实用性：

- 用户/供应商/模型/请求记录删除（二次确认）
- 批量删除支持
- 用户/供应商/模型/请求记录列表导出（CSV/Excel）
- 导出进度提示

### v3.1: 响应式 + 主题 + 性能

完成响应式设计、主题切换和性能优化，提升整体用户体验：

- 移动端/平板端适配、侧边栏折叠功能、移动端菜单抽屉
- 明/暗主题切换、主题持久化存储、跟随系统主题
- 修改 Token 功能、Token 复制快捷方式、操作日志展示
- 列表虚拟滚动、组件按需加载、缓存策略优化
- 快捷键支持、面包屑导航、返回顶部按钮、全局搜索

### v4.0: 计费管理

实现用户余额管理和模型计费功能，支持商业化运营：

- 模型价格配置（输入 Token 价格、输出 Token 价格）
- 用户余额展示、余额调整（充值/扣减）
- 余额管理与充值记录整合页面（左侧导航切换）
- 充值记录查询、按用户名/时间筛选、分页功能
- 余额为空自动停止服务

## 技术栈

| 类别 | 技术 | 版本 |
|------|------|------|
| 前端框架 | Vue | 3.x |

| 构建工具 | Vite | 5.x |
| UI 组件库 | Ant Design Vue | 4.x |
| 路由 | Vue Router | 4.x |
| 状态管理 | Pinia | 2.x |
| HTTP 客户端 | Axios | 1.x |
