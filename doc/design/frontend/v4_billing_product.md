# v4.0 计费管理 - 前端产品文档

## 模块概述

实现用户余额管理和模型计费功能的前端界面，支持商业化运营。管理员可以为模型设置价格、调整用户余额、查看充值记录。

## 功能模块

### 1. 余额管理页面

查看用户余额、充值/扣减余额、查看充值记录，整合在一个页面中，左侧导航切换。

**页面布局**

```
+----------------------------------------+
|  余额管理                               |
+----------------------------------------+
| [余额管理] [充值记录]                   | ← 左侧导航
+----------------------------------------+
|                                        |
|   余额管理 Tab 内容                     |
|   - 用户列表（含余额）                   |
|   - 充值/扣减操作                       |
|                                        |
+----------------------------------------+
```

**余额管理 Tab**

**界面元素**
- 用户列表余额展示
  - 表格展示：用户名、余额、操作
  - 充值按钮
  - 扣减按钮

- 余额调整对话框
  - 用户名称（只读）
  - 当前余额（只读）
  - 调整类型选择器（充值/扣减）
  - 调整金额输入框
  - 备注输入框（可选）
  - 确认/取消按钮

**交互流程**
1. 在用户列表页，点击"充值"或"扣减"按钮
2. 输入调整金额和备注
3. 确认后更新余额
4. 操作成功后提示并刷新列表

**API 调用**
- `GET /user/list.json` - 获取用户列表（含余额）
- `POST /user/:id/balance/adjust.json` - 调整用户余额

---

**充值记录 Tab**

**界面元素**
- 充值记录列表
  - 表格展示：用户名、充值金额、充值后余额、备注、操作时间
  - 搜索框（按用户名搜索）
  - 筛选器（按时间范围筛选）
  - 分页器

**交互流程**
1. 点击左侧导航"充值记录"切换到记录列表
2. 按需筛选或搜索记录
3. 支持分页浏览历史记录

**API 调用**
- `GET /balance/recharge/list.json` - 获取充值记录列表

---

### 2. 模型价格配置

为每个模型设置输入和输出 Token 的计费价格，价格存储在模型对象中。

**界面元素**
- 模型列表价格展示
  - 表格新增列：输入价格、输出价格
  - 编辑按钮

- 编辑模型对话框
  - 模型名称输入框
  - 供应商选择器
  - 启用状态开关
  - 输入 Token 价格输入框（单位：元/千 Token）
  - 输出 Token 价格输入框（单位：元/千 Token）
  - 保存/取消按钮

**交互流程**
1. 在模型管理页面，点击"编辑"按钮
2. 弹出编辑对话框，自动填充当前信息（包括价格）
3. 修改信息后保存
4. 保存成功后提示更新完成

**API 调用**
- `GET /model/list.json` - 获取模型列表（含价格）
- `PUT /model/:id` - 更新模型（包含价格）
