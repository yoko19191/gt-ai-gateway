# v2.0/v2.1 前端产品文档

## 版本概述

实现监控分析功能，包括请求记录查看、仪表盘统计和 API 测试工具。

---

## 功能模块

### 请求记录（v2.0）

**界面元素**
- 请求记录列表页面
  - 表格展示：ID、用户名称、供应商名称、模型名称、状态、创建时间
  - 搜索框（按用户名、模型名搜索）
  - 筛选器（按状态、时间范围筛选）
  - 刷新按钮
  - 分页器

- 请求详情对话框
  - 基本信息：请求 ID、用户、供应商、模型、状态、时间信息
  - 请求数据（JSON 格式，可折叠）
  - 响应数据（JSON 格式，可折叠）
  - 复制数据按钮

**交互流程**
1. 请求记录页展示历史请求（默认倒序）
2. 支持按时间范围、状态筛选
3. 点击某条记录查看详情
4. 请求数据和响应数据以 JSON 格式展示

---

### 仪表盘增强（v2.1）

**界面元素**
- 统计卡片
  - 今日请求总数
  - 今日请求成功率
  - 活跃用户数
  - 活跃模型数

- 最近请求列表
  - 快捷跳转到请求详情

- 自动刷新控制
  - 刷新按钮
  - 暂停/开始刷新按钮
  - 最后更新时间显示

**交互流程**
1. 进入系统后默认展示仪表盘
2. 数据自动刷新（30秒间隔）
3. 点击"查看更多"跳转到请求记录页面

---

### API 测试工具（v2.1）

**界面元素**
- 测试面板
  - API 格式选择器（OpenAI/Anthropic）
  - 模型选择器（从已有模型中选择）
  - 消息输入区域（支持多轮对话）
  - 参数配置（temperature、max_tokens 等）
  - 发送按钮
  - 响应展示区域

- 历史记录面板
  - 测试历史列表
  - 加载历史配置
  - 删除历史记录

**交互流程**
1. 选择 API 格式和模型
2. 输入对话消息
3. 配置可选参数
4. 点击发送
5. 展示流式响应或完整响应
6. 支持"清除"和"重新发送"
