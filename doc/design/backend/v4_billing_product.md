# v4.0 计费管理 - 后端产品文档

## 模块概述

实现用户余额管理和模型计费功能的后端 API，支持商业化运营。包括模型价格配置、余额调整、充值记录和自动停止服务。

---

## 功能模块

### 1. 模型价格配置

为每个模型设置输入和输出 Token 的计费价格。

**功能说明**
- 模型价格直接存储在模型对象中
- 支持设置输入 Token 价格和输出 Token 价格
- 价格单位为元/千 Token

**接口列表**
- 获取模型列表（含价格）
- 更新模型（包含价格）

---

### 2. 用户余额管理

查看用户余额、充值、扣减余额。

**功能说明**
- 用户余额直接存储在用户对象中
- 支持充值和扣减操作
- 充值操作会记录到充值记录表
- 扣减操作不记录（用于消费扣减）

**接口列表**
- 获取用户列表（含余额）
- 调整用户余额（充值/扣减）

---

### 3. 充值记录

查询用户余额的充值记录，仅记录充值操作。

**功能说明**
- 仅记录充值操作，不保留扣减记录
- 支持按用户名搜索
- 支持按时间范围筛选
- 支持分页查询

**接口列表**
- 获取充值记录列表

---

### 4. 自动停止服务

用户余额为空时，自动停止该用户的 API 服务。

**功能说明**
- 余额 ≤ 0 时，该用户的所有 API 请求返回错误
- 错误信息提示余额不足
- 用户充值后，服务自动恢复

**错误响应**
```json
{
  "code": 402,
  "message": "余额不足，请充值后再试",
  "data": {
    "current_balance": 0
  }
}
```
